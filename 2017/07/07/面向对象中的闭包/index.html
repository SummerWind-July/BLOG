闭包

概念

闭包指的就是那些能够访问独立的变量的函数！

闭包要解决的问题

函数中声明的变量不能被函数外部直接使用

闭包的原理

闭包的作用

  1. 给函数提供一个私有的变量
  2. 保护变量，给变量的设置提供专有的渠道，在这个渠道中可以添加一些校验的逻辑

递归实现的斐波那契数列存在的性能问题

由于存在大量的重复的计算，所以导致在40多之后，就算不出来了

解决方案

  1. 提供一个缓存，用这个缓存来存储计算出来的数据
  2. 每次在计算的时候，首先先从缓存中去获取,如果有，就直接返回
  3. 如果没有，就通过递归的方式去计算
  4. 计算出来之后，一定要记着保存到缓存中去，以便下次使用
